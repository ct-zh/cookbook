# 辣椒炒肉

#### 原料

- 猪肉:  建议五花肉+里脊肉混搭, 想肥一点就多一点五花; 也可以直接买梅花肉 ( 猪的上肩肉, 肥瘦相间 ); 
- 辣椒: 最佳搭配樟树港辣椒, 其次长沙本地青椒; 螺丝椒也可以;
- 配菜: 大蒜叶 / 豆豉 
- 配料: 一定要有蒜

#### 准备

- 猪肉切稍大一点的丝, 五花肉要切肥瘦相间; 加三勺生抽、两勺蚝油、一勺油拌匀;  不嫌麻烦的话五花和里脊分开装, 肥肉和瘦肉分开装
- 辣椒去头切滚刀块, 小辣椒直接拍碎撕开; 
- 蒜拍碎切丁, 豆豉提前洗净晾干, 大蒜叶切片;

#### 烹饪

- ( 这步可省略 ) 如果想吃辣一点的辣椒, 锅洗干净烧干水, 放入辣椒煸热, 最好煸到辣椒边缘有点焦焦的感觉; 放入一点点盐拌匀, 这个时候应该已经辣得呛人了; 直接装盘;
- 热锅冷油
- ( 这步可省略 ) 先下三分之一蒜, 爆香之后下 肉中肥的那一部分, 开大火爆炒, 炒出汁水后只把肉装盘;
- 倒入辣椒, 炒出青椒的虎皮, 如果做了上一步, 则让辣椒稍微吸收一下肉的汁水, 一勺生抽沿锅边淋入; 如果没有, 两勺生抽沿锅边淋入;
- 因为肉腌制过, 这里辣椒要单独炒出盐味, 生抽口味不重的可以额外放一点点盐; 最好能炒出虎皮炒熟; 
- 下剩下三分之一蒜, 倒入剩下的肉、大蒜叶、豆豉( 没有就不用倒了 ) 加一勺老抽, 大火翻炒把肉炒熟, 然后倒入剩下的蒜, 再翻炒两下出锅装盘;
